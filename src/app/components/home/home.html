Polari Research Framework

A generalizable research framework that modularizes open-source Python libraries like PyTorch, FEniCS, and other scientific frameworks. Create custom classes through an intuitive interface, design tables and views for your objects, and leverage state-objects for no-code programming with dynamically defined states.
